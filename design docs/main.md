设计文档：千王之王

一个roguelite原型，参考了小丑牌的部分设计结构。
1. 概述
    玩家在游戏开始时获得初始骰子，通过不断赢下对局来升级骰子，挑战更难关卡。
    目标是，从初始的骰子，到养成一组想要任何点数都能大概率投出的千王之骰。
2. rogue要素
    2.1 局外养成
        获得更多的初始骰子。初始骰子可以影响骰子面数、骰子概率等。
    2.2 局内升级：出千
        骰子升级：除了面数外，概率、数字等都会发生变化。
        拨动骰子：在骰子运转中，通过隐秘地拨动骰子，以实现想要达成的结果。
        <!-- 消耗品：一次性使用，效果包括重新投掷骰子、投掷前获取额外数字加成。 -->
        获得新骰子：单次可以丢出更多的骰子。得分计算一般为骰子之和，但是也有倍率骰。
3. 游戏机制
    3.1 游戏流程
        本游戏由一系列关卡与出千流程交替组成。
        关卡流程中，玩法需要在有限次投掷中，获得目标点数。
        出千流程中，升级骰子，或者获得新骰子。
        3.1.1  关卡流程
            每个关卡都有一个目标点数，玩家需要在有限次投掷中，获得目标点数。
            一次投掷，最少使用1个骰子，最多使用3个骰子。玩家初始只有1个骰子，额外骰子通过出千流程获得。
            通过关卡时，获得积分。积分的计算方式与关卡、剩余投掷次数相关。
        3.1.2 出千流程
            消耗积分，来升级骰子，或者购买新骰子。
            升级为当局永久性提升。
            骰子总上限为5个，但是单个关卡只能最多使用3个，且每个关卡一旦投掷了第一次，后续投掷次数就不能改变骰子。
            当玩家手中骰子数量达到上限时，后续出千流程想要购买新骰子时，必须出售已有骰子，腾出空位。
4. 游戏设计
    4.1 骰子设计
        4.1.1 初始骰子：玩家在关卡开始时可以选择的骰子，只能择一开始游戏
            a. 标准六面骰
            b. 标准八面骰
            c. 灌了铅的六面骰。其中1，6的面数灌铅，出现概率权重乘以3.
        4.1.2 额外骰子：只能在出千流程中购买
            a. 四面骰：数字为1,2,3,4，概率均等
            b. 六面骰：数字为1，1，1，6，6，6 概率均等
            c. 六面倍率骰。分别为0.8,0.9,1.1,1.2,1.3,1.4 每个数字出现概率均等。该骰子在计算积分时，会先计算其他骰子的总和，再乘以倍率。多个倍率骰子的倍率分别相乘。
            d. 空白骰子。会复制其他骰子的最大数字。在计算积分时，会将空白骰子的数字，替换为其他骰子的最大数字。
    4.2 升级设计
        4.2.1 升级成本
            每个骰子升级，需要消耗一定的积分。
            升级成本与骰子面数、骰子类型相关。
        4.2.2 升级效果
            a. 增加最大数字的概率权重为原有的两倍
            b. 将一个随机面替换为其他某个面的数字
            c. 每个面的数字+1.倍率骰无效
    4.3 消耗品设计
        消耗品也可以在出千流程中购买，但是每个消耗品只能用1次，同时最多拥有3个消耗品。
        4.3.1 回溯祝福
            消耗1个消耗品，重新投掷1次骰子，不额外扣除投掷次数。
        4.3.2 贪心祝福
            下次投掷，最大点数额外+1，如果有重复的，都加1.
        4.3.3 穷鬼祝福
            下次计分，会忽略掷出最小值的骰子。
    4.3 关卡设计
        4.3.1 目标点数
            每个关卡都有一个目标点数，玩家需要在有限次投掷中，获得目标点数。
        4.3.2 积分计算
            每个关卡通关时，获得积分为：
            基础胜利积分+剩余投掷次数*2
4. 界面设计
    4.1 主界面
        主界面包括游戏标题、游戏描述、骰子选择、开始游戏按钮。
    4.2 关卡界面
        关卡界面包括返回主菜单按钮、关卡信息显示、骰子显示、投掷按钮、消耗品显示、新骰子显示。
        投掷按钮点击后，会显示当前投掷的结果，以及当前积分。
    4.3 出千界面
        出千界面包括返回主菜单按钮、可购买骰子、可购买的升级、下一关的目标点数。
